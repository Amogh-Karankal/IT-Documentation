# IT Terminology Glossary

Common terms you'll encounter in IT support.

---

## A

**Active Directory (AD)**
Microsoft's directory service for Windows networks. Stores user accounts, computers, groups, and policies. Think of it as a database of everyone and everything in the company.

**Admin / Administrator**
Someone with elevated permissions to manage systems, create accounts, install software, etc.

**Antivirus / AV**
Software that detects and removes malicious software (malware).

**Authentication**
Proving you are who you say you are (usually with username + password + MFA).

**Authorization**
What you're allowed to do after authentication. Permissions.

**Azure**
Microsoft's cloud computing platform. Hosts services, virtual machines, and applications.

---

## B

**Backup**
A copy of data stored separately for recovery purposes.

**Bandwidth**
The amount of data that can be transferred over a network. More bandwidth = faster speeds.

**BIOS**
Basic Input/Output System. Firmware that initializes hardware when computer starts.

**BitLocker**
Microsoft's disk encryption feature. Protects data if laptop is stolen.

**Bluetooth**
Wireless technology for short-range device connections (headphones, mice, etc.).

**Brute Force**
Attack method that tries every possible password combination until one works.

---

## C

**Cache**
Temporary storage to speed up repeated access. Clearing cache often fixes issues.

**Certificate**
Digital credential that verifies identity. Used for HTTPS, VPN, email signing.

**Cloud**
Services and storage hosted on remote servers (Azure, AWS, Google Cloud) instead of local hardware.

**Conditional Access**
Security policy that controls access based on conditions (location, device, risk level).

**Credentials**
Username and password (and sometimes MFA) used to log in.

---

## D

**DHCP**
Dynamic Host Configuration Protocol. Automatically assigns IP addresses to devices on a network.

**DNS**
Domain Name System. Translates website names (google.com) to IP addresses.

**Domain**
A network of computers managed together, usually with Active Directory.

**Driver**
Software that allows the operating system to communicate with hardware (printer driver, graphics driver).

---

## E

**Encryption**
Converting data into unreadable format that requires a key to decrypt.

**Endpoint**
Any device that connects to a network (laptop, phone, tablet).

**Entra ID**
Microsoft's cloud identity service (formerly Azure AD). Manages user accounts for Microsoft 365.

**Escalation**
Passing a support ticket to a higher tier with more expertise.

---

## F

**Firewall**
Security system that monitors and controls network traffic based on rules.

**Firmware**
Software programmed into hardware devices (router firmware, BIOS).

**FQDN**
Fully Qualified Domain Name. Complete address like `server.company.local`.

---

## G

**Gateway**
Device that connects different networks (usually the router).

**Group Policy (GPO)**
Settings applied to users and computers in Active Directory.

---

## H

**Hardware**
Physical components (laptop, monitor, keyboard, server).

**Help Desk**
First point of contact for IT support.

**Hostname**
The name assigned to a computer on a network.

**HTTPS**
Secure version of HTTP. The "S" means encrypted connection.

---

## I

**Identity**
Digital representation of a user (username, account, profile).

**Image**
A complete copy of a system's software configuration. Used to deploy identical setups.

**Incident**
An unplanned event that disrupts service (outage, error, failure).

**IP Address**
Unique number identifying a device on a network (e.g., 192.168.1.100).

**ISP**
Internet Service Provider. Company that provides internet access.

**ITIL**
IT Infrastructure Library. Framework for IT service management best practices.

---

## J

**Jira**
Popular ticketing/project management software.

---

## K

**KB / Knowledge Base**
Collection of articles documenting known issues and solutions.

---

## L

**LAN**
Local Area Network. The network within a building or office.

**LDAP**
Lightweight Directory Access Protocol. Protocol for accessing directory services like AD.

**License**
Permission to use software. Usually purchased per user or device.

**Local**
On the device itself, not on a network or cloud.

**Log**
Record of events, errors, and activities. Essential for troubleshooting.

---

## M

**Malware**
Malicious software (viruses, ransomware, spyware, trojans).

**MDM**
Mobile Device Management. Software to manage and secure phones/tablets.

**MFA / 2FA**
Multi-Factor Authentication / Two-Factor Authentication. Extra verification beyond password.

**Microsoft 365 (M365)**
Microsoft's cloud productivity suite (Outlook, Teams, Word, Excel, etc.).

---

## N

**NAT**
Network Address Translation. Allows multiple devices to share one public IP.

**Network**
Connected devices that can communicate with each other.

**NIC**
Network Interface Card. Hardware that connects a computer to a network.

---

## O

**OAuth**
Authorization protocol. How you "Sign in with Google" on other websites.

**On-premises / On-prem**
Hosted locally at the company's location, not in the cloud.

**OS**
Operating System (Windows, macOS, Linux).

**Outage**
When a service or system is unavailable.

---

## P

**Patch**
Software update that fixes bugs or security vulnerabilities.

**Permission**
Access rights granted to users (read, write, delete, admin).

**Phishing**
Fraudulent attempt to steal credentials or data via fake emails/websites.

**Ping**
Network command that tests connectivity to another device.

**PKI**
Public Key Infrastructure. System for managing digital certificates.

**Port**
Numbered endpoint for network communication (HTTP uses port 80, HTTPS uses 443).

**PowerShell**
Microsoft's command-line shell and scripting language.

**Privilege**
Level of access. "Elevated privileges" = admin rights.

---

## Q

**Queue**
List of tickets waiting to be worked on.

---

## R

**RAM**
Random Access Memory. Temporary memory for running programs. More RAM = better multitasking.

**Ransomware**
Malware that encrypts files and demands payment to unlock.

**RDP**
Remote Desktop Protocol. Microsoft's remote access technology.

**Registry**
Windows database storing system and application settings.

**Remote**
Accessing something from a different location (remote desktop, remote support).

**Restore**
Recovering data from a backup.

**Root Cause**
The underlying reason why a problem occurred.

---

## S

**SaaS**
Software as a Service. Cloud-based software you subscribe to (Microsoft 365, Salesforce).

**SAML**
Security Assertion Markup Language. Standard for exchanging authentication data. Used for Single Sign-On.

**SAN**
Storage Area Network. High-speed network providing storage access.

**Screenshot**
Captured image of what's on a screen.

**Server**
Computer that provides services to other computers (file server, email server).

**ServiceNow**
Popular IT service management (ITSM) platform.

**SLA**
Service Level Agreement. Promised response and resolution times.

**Software**
Programs and applications (Word, Chrome, Windows).

**Spam**
Unwanted email, often advertising or phishing.

**SSD**
Solid State Drive. Faster storage than traditional hard drives.

**SSO**
Single Sign-On. Log in once to access multiple applications.

**SSL/TLS**
Protocols that encrypt network communication (used by HTTPS).

**Subnet**
A logical division of a network.

**Sync**
Keeping data consistent across multiple devices or locations.

---

## T

**TCP/IP**
Core protocols of the internet. How data travels between computers.

**Ticket**
A record of a support request or incident.

**Token**
A physical device or software code used for authentication.

**Troubleshooting**
Process of identifying and resolving problems.

---

## U

**UAC**
User Account Control. Windows security feature that prompts for admin confirmation.

**UPN**
User Principal Name. Usually the email address used to log in.

**URL**
Web address (https://www.example.com).

**USB**
Universal Serial Bus. Common connector for devices.

---

## V

**VDI**
Virtual Desktop Infrastructure. Desktops hosted on servers, accessed remotely.

**Virtual Machine (VM)**
Software-based computer running on a physical server.

**Virus**
Type of malware that spreads by attaching to files.

**VPN**
Virtual Private Network. Secure connection to company network from remote location.

---

## W

**WAN**
Wide Area Network. Network spanning large geographic area (internet is a WAN).

**Wi-Fi**
Wireless networking technology.

**Windows Hello**
Biometric login for Windows (fingerprint, face recognition).

**Workstation**
A user's computer, typically a desktop or laptop.

---

## Z

**Zero Trust**
Security model that trusts nothing by default. "Never trust, always verify."

**ZIP**
Compressed file format. Reduces file size.
