# New Employee IT Setup Checklist

Complete guide for onboarding new employees.

---

## 📋 Overview

When a new employee joins, IT needs to set up everything they need on **Day 1**. This checklist ensures nothing is missed.

---

## ⏰ Timeline

| When | What |
|------|------|
| **1 week before** | Receive onboarding request from HR |
| **3 days before** | Create accounts, order equipment |
| **1 day before** | Configure laptop, test everything |
| **Day 1** | Meet with employee, hand off equipment |
| **Day 1 + 1 hour** | Follow up — everything working? |
| **Week 1** | Check in — any issues? |

---

## 📝 Pre-Arrival Setup (IT does this before Day 1)

### Account Creation

```
☐ Create user account in Entra ID / Active Directory
   - Username: firstname.lastname@company.com
   - Temporary password generated
   - Set "must change password on first login"

☐ Assign to correct groups
   - Department group (HR, IT, Finance, etc.)
   - Location group (if applicable)
   - Distribution lists

☐ Assign licenses
   - Microsoft 365 E3/E5 license
   - Any department-specific licenses

☐ Configure mailbox
   - Email address active
   - Add to required distribution lists
   - Set up shared mailbox access (if needed)
```

### Equipment Preparation

```
☐ Laptop/Desktop assigned
   - Asset tag recorded
   - Assigned to user in inventory system

☐ Operating system configured
   - Windows 11 installed and updated
   - Joined to domain / Entra ID
   - BitLocker encryption enabled

☐ Standard software installed
   - Microsoft Office / M365 Apps
   - Company VPN client
   - Antivirus/endpoint protection
   - Teams / Slack
   - Web browsers (Chrome, Edge)
   - Department-specific software

☐ Peripherals ready (if applicable)
   - Monitor(s)
   - Keyboard and mouse
   - Headset
   - Docking station
   - Webcam
```

### Access Setup

```
☐ VPN access configured
   - VPN profile installed
   - Test connection

☐ Network access
   - Wi-Fi credentials ready
   - Wired network port assigned (if office)

☐ Printer access
   - Default printers installed
   - Test print

☐ File access
   - Network drive mappings
   - SharePoint sites
   - Shared folders
```

## 📆 Day 1 — Employee Arrival

### Welcome Meeting (30-60 minutes)

```
☐ Introduce yourself
   "Hi, I'm [Name] from IT. I'll be getting you set up today."

☐ Hand over equipment
   - Laptop/desktop
   - Peripherals
   - Any access cards/badges

☐ First login
   - Help user log in with temp password
   - Guide through password change
   - Set up Windows Hello / biometrics (optional)

☐ MFA enrollment
   - Install Microsoft Authenticator app
   - Register phone for MFA
   - Test MFA is working

☐ Email setup
   - Verify Outlook is working
   - Sync email on mobile device (if requested)
   - Show calendar basics

☐ Teams/Communication setup
   - Log into Teams
   - Test audio/video
   - Show how to chat and call

☐ VPN demonstration
   - Show how to connect
   - Test accessing internal resources

☐ Quick orientation
   - How to submit IT tickets
   - IT support contact info
   - Self-service password reset
```

## 🚨 Common Day 1 Issues

| Issue | Quick Fix |
|-------|-----------|
| Can't log in | Check username spelling, check Caps Lock |
| MFA not working | Ensure phone time is correct (auto), reinstall Authenticator |
| Outlook won't connect | Check license assigned, wait 15 mins for propagation |
| VPN fails | Check internet first, verify credentials |
| No Wi-Fi | Ensure correct SSID and password |

## 💡 Tips for Smooth Onboarding

1. **Prepare early** — Don't wait until Day 1
2. **Test everything** — Log in as user before they arrive
3. **Be patient** — It's overwhelming for new employees
4. **Follow up** — Check in after a few days
5. **Document** — Log everything in the ticket
